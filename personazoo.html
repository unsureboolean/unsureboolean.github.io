About PersonaZoo

PersonaZoo is an interactive chat experience that lets you converse with AI personas based on fictional characters and archetypes. Each persona has its own unique personality, speech patterns, and knowledge base.

This project uses OpenAI's API to generate responses that match the selected persona's character. The personas are designed to be entertaining and engaging, providing a fun way to interact with AI.

Created by Jeff Tuel as a demonstration of AI character simulation and interactive web experiences.
